## 🎮 Quick Controls Reference

### Essential Controls
- **Click Canvas**: Lock pointer for first-person control
- **WASD / Arrows**: Move around
- **Mouse**: Look around
- **F**: Toggle flying mode
- **Space**: Fly up (in flying mode)
- **Shift**: Fly down (in flying mode)
- **M**: Open Streamlit menu
- **C**: Open NPC controls
- **ESC**: Close menus

### NPC Management
- **C Key**: Open NPC control panel
- **Meeting Button**: Gather all 30 agents at center
- **Tables Button**: Send agents to workstations
- **Individual Toggles**: Activate/deactivate specific agents

## 🔧 Customization Guide

### Change Avatar URLs
Edit `components/ready-player-me-config.ts`:
```typescript
export const READY_PLAYER_ME_AVATARS = [
  {
    id: "avatar_1",
    name: "Agent CEO",
    avatarUrl: "YOUR_READY_PLAYER_ME_URL.glb",
  },
  // ... more avatars
]
```

### Update Streamlit URLs
Edit `components/virtual-gallery.tsx`:
```typescript
const GALLERY_ITEMS = [
  {
    title: "Your App",
    streamlitUrl: "https://entremotivator.com/npc-shortcode-windows/",
    // ... position and rotation
  },
]
```

### Modify NPC Behavior
Edit `DEFAULT_NPC_DATA` in `components/virtual-gallery.tsx`:
```typescript
{
  id: 1,
  name: "Agent Name",
  speed: 0.5, // Adjust speed
  dialogue: ["Your custom dialogue"],
  // ... other properties
}
```

## 🐛 Troubleshooting

### Common Issues

**NPCs not appearing?**
- Check browser console for errors
- Verify Ready Player Me URLs are accessible
- Try refreshing the page

**Streamlit apps not loading?**
- Ensure apps are running on port 8501
- Check CORS settings
- Verify URLs are correct

**Performance issues?**
- Open NPC controls (Press C)
- Deactivate some agents
- Close unused Streamlit apps
- Reduce browser window size

**Build errors?**
- Run `pnpm install` to ensure dependencies
- Clear `.next` folder and rebuild
- Check Node.js version compatibility
